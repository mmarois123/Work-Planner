Only use `@delegated(Irina)` — she is the user's direct report and the only person tasks are delegated to.

Tasks mentioning LJ (LittleJohn PE owners), Bob (CFO, user's boss), or other colleagues are work the user needs to deliver *to* them, not tasks delegated *to* them. Do not add `@delegated()` tags for these people.

**Why:** The user's role is FP&A — Irina reports to them. Everyone else (Bob, LJ, Ron, Gary, etc.) are stakeholders or leadership the user serves.

**How to apply:** When routing new tasks, only apply `@delegated(Irina)` for Irina items. For all others, place in the appropriate section (Bob Review, General, etc.) without delegation tags.
